# GramSathi AI – Intelligent Rural Development Assistant

GramSathi AI is an interactive Single Page Web Application designed to empower rural communities. It provides farmers, students, rural entrepreneurs, self-help groups (SHGs), NGOs, and Panchayat members with reliable guidelines, calculators, project proposal frameworks, and an AI chat assistant.

---

## 🌟 Key Features

1. **Personalization & Onboarding Wizard:**
   - Adapts to user type (Farmer, Entrepreneur, NGO, Official, Student).
   - Tailors weather readings, crop recommendations, and business suggestions based on state, district, budget, and land ownership.
   
2. **Conversational Assistant (GramSathi AI Chat):**
   - Simulated conversational interface configured with rural databases.
   - Outputs answers structured into the standard **10-Part Response Format** (Summary, Detailed Explanation, Benefits, Challenges, Step-by-Step, Example, Best Practices, Government Support, Common Mistakes, Recommendations).

3. **Government Welfare schemes catalog:**
   - Searchable, category-filtered list of critical central schemes (PM-KISAN, PMFBY, PMAY-G, Kisan Credit Card, Mudra Loan, Ayushman Bharat).
   - Explains Purpose, Eligibility, Benefits, Documents, Application process, Deadlines, and FAQs in detail.

4. **Farm & Climate Hub:**
   - Advises on crop options based on soil type and irrigation facilities.
   - Contains a comparative decision tool for Drip vs Sprinkler systems.

5. **Rural Entrepreneurship Planner:**
   - Startup calculators and SWOT profiles for highly viable rural businesses (Dairy Farm, Poultry, Mushroom Cultivation, Solar Business, Food Processing).
   - Estimates capital requirements, monthly profit, risks, and displays government subsidies.

6. **Panchayat & NGO Proposal Generator:**
   - Compiles formal project proposals (Water infrastructure, Solid Waste Management, Solar utilities, Skill center grants) based on budget, village name, and duration.
   - Single-click clipboard copying and browser print layouts.

7. **Disaster Resilience Center:**
   - Active guides for floods, cyclones, droughts, and pest outbreaks, split by chronological phases (Before, During, After).

8. **Rich Design System & Accessibility:**
   - Sleek and vibrant agricultural-themed dark/light theme templates.
   - Built-in multi-language dropdown support.

## 🚀 How to Run Locally

Since this is a client-side Single Page Application using standard HTML5, Vanilla CSS, and modern ES6 JavaScript, there are no build steps or heavy dependencies required.

1. **Option A (Double Click):**
   Simply open [index.html](index.html) in any modern web browser (Chrome, Firefox, Safari, Edge).

2. **Option B (Local Dev Server):**
   If you have Node.js installed, you can run a simple server:
   ```bash
   npx serve .
   ```
   Or using Python:
   ```bash
   python3 -m http.server 8000
   ```
   Open `http://localhost:8000` in your web browser.
